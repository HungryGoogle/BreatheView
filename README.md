# BreatheView
呼吸灯闪烁效果的自定义View


##如何使用


Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}Copy
Step 2. Add the dependency

	dependencies {
	        compile 'com.github.jianfeng318:BreatheView:0.1.2'
	}
